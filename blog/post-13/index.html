1.setRandom() 是計算的method取整數 (int)(Math.random()*(最大範圍值-最小範圍值+1)+最小範圍值)

2.show() method 印出取到亂數整數

3.sort_setRandom() method 顧名思義就是排序亂數值

4.main() method 搭配USER 使用Scanner 取出需求亂數

import java.util.*;
public class Bonus_line
{
	//random的範圍設定
	int numtime;
	public int[] setRandom(int[] num,int numlength,int maxtime,int mintime)
	{
		this.numtime=numlength;
		/*this.num=num;
		this.numlength=numlength;
		this.maxtime=maxtime;
		this.mintime=mintime;*/
		for(int i=0;i<this.numtime;i++)
		{
			num[i]=(int)(Math.random()*(maxtime-mintime+1)+mintime);
			int j=0;
			while(j<i)
			{
				if(num[j]==num[i])
				{
					i--;
					break;
				}
				j++;
			}
		}
		return num;
	}
	
	public static void show(int[] num)
	{
		for(int i=0;i<num.length;i++)
			System.out.print(num[i]+"\t");
	}
	public void sort_setRandom(int[] num)
	{
		Arrays.sort(num);
	}
	public static void main(String args[])
	{
		int numlength,minnum,maxnum;
		int[] num;
		System.out.print("請輸入要幾個亂數:");
		Scanner sc=new Scanner(System.in);
		numlength=sc.nextInt();
		num= new int[numlength];
		do
		{
		
		
		System.out.print("請輸入上界值:");
		maxnum=sc.nextInt();
		
		System.out.print("請輸入下界值:");
		minnum=sc.nextInt();
		}while(maxnum<minnum && maxnum-minnum<numlength);
		Activity1_23 acti=new Activity1_23();
		acti.setRandom(num,numlength,maxnum,minnum);
		System.out.print("中獎的數字為:");
		show(num);	
		acti.sort_setRandom(num);
		System.out.println();
		System.out.print("排序後大到小為: ");
		show(num);
		
		
		
	}
}

執行方式

在目標路徑存成 Bonus_line.java

  1. win10開啟命令提示字元、Mac 開啟 terminal

  2. 在Bonus_line.java目標路徑，執行:javac Bonus_line.java

  3. 產生一個Bonus_line.class，即可執行:java Bonus_line

  4. 或使用相關IDE 能免去上面2、3步驟麻煩，直接執行
